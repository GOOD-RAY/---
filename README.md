# 俄乌冲突对国际粮食期货价格的异质性影响
> 基于分位数回归的实证研究 | 北京航空航天大学 数理统计探索性大作业（选题T4）

## 项目简介
本项目以 CBOT 大豆、玉米、小麦三大粮食期货为研究对象，采用分位数回归方法，结合 Bootstrap 自助标准误、联合 Wald 检验、修正 Chow 检验等统计工具，实证分析俄乌冲突对粮食期货价格在不同分位点的异质性冲击效应，突破传统 OLS 均值回归的局限，完整刻画地缘冲突对价格全分布的影响。

## 研究背景
俄乌两国合计占全球小麦出口 33%、玉米出口 16%，是全球粮食供给的核心主体。2022 年 2 月俄乌冲突爆发后，国际粮食价格短期剧烈波动，全球粮食安全风险显著上升。

传统均值回归仅能刻画冲突的平均冲击，无法区分低价、中价、高价区间的差异化影响。本项目从条件分布全视角出发，回答核心问题：
> 俄乌冲突对不同价位的粮食期货价格，冲击强度是否存在显著差异？

## 数据说明
### 数据来源
所有数据均来自 Investing.com 公开数据库，为日度收盘价数据：
- 被解释变量：CBOT 大豆、玉米、小麦期货收盘价
- 控制变量：WTI 原油期货价格、美元指数（DXY）

### 样本区间
2019 年 1 月 1 日 — 2025 年 12 月 31 日，经日期匹配与缺失值剔除后，每个品种共 1760 个有效日度观测。

### 变量定义
| 变量名 | 变量说明 |
|--------|----------|
| `log_price` | 期货收盘价取自然对数，模型被解释变量 |
| `conflict` | 冲突虚拟变量，2022-02-24 及之后取 1，否则取 0，核心解释变量 |
| `log_oil` | WTI 原油收盘价取自然对数，控制生产成本传导效应 |
| `log_dxy` | 美元指数取自然对数，控制美元计价效应 |
| `trend` | 线性时间趋势项，剥离价格长期趋势因素 |
| `month` | 月份虚拟变量，控制农业生产周期带来的季节性波动 |
